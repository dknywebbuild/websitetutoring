# Pinetree Tutoring Website

Static site for **Pinetree Tutoring**. Built for GitHub Pages.

## Files
- `index.html` — main page
- `style.css` — styling
- `script.js` — optional JS (currently empty)
- `404.html` — custom not-found page
- `.nojekyll` — disables Jekyll processing (recommended for plain HTML sites)
- `robots.txt` and `sitemap.xml` — basic SEO files
- `CNAME` — optional custom domain file (edit or delete if not using a domain)

## Publish on GitHub Pages
1. Create a GitHub repo (public).
2. Upload all files from this ZIP to the repo root.
3. Go to **Settings → Pages**.
4. Under **Source**, choose:
   - Branch: `main`
   - Folder: `/ (root)`
5. Save. Your site will be live in a minute.

**If using a custom domain:**
- Edit `CNAME` to your domain (example: `www.pinetreetutoring.com`) and keep it in the repo root.
- Make sure your DNS records are set in your domain provider.
